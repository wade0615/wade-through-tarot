## 文案撰寫指南

### 內容要求

每個區塊應包含以下元素：

#### loveReading (愛情解讀) 150~200 字
1. **正位意義** (2-3 句): 這張牌在愛情中的正面含義
2. **逆位提醒** (1-2 句): 需要注意的警訊
3. **單身者指引** (1-2 句): 對單身者的建議
4. **有伴侶者指引** (1-2 句): 對有伴侶者的建議
5. **行動建議** (1 句): 具體可執行的建議

#### careerReading (事業解讀) 150~200 字
1. **正位意義** (2-3 句): 這張牌在事業中的正面含義
2. **逆位提醒** (1-2 句): 需要注意的警訊
3. **求職者指引** (1-2 句): 對求職者的建議
4. **在職者指引** (1-2 句): 對在職者的建議
5. **財務提示** (1 句): 相關財務建議

#### healthReading (健康解讀) 150~200 字
1. **身體健康** (2-3 句): 身體層面的健康提示
2. **心理健康** (2-3 句): 心理/情緒層面的建議
3. **生活習慣** (1-2 句): 日常作息、飲食建議
4. **注意事項** (1 句): 特別需要關注的健康面向

### 撰寫風格

- 使用繁體中文
- 語氣專業但親切
- 避免過於絕對的預測用語
- 強調「指引」而非「預言」
- 結合牌面本身的象徵意義

### 範例格式

```typescript
loveReading: "【正位】這張牌代表...(2-3句)。【注意】當逆位時...(1-2句)。【單身者】建議你...(1-2句)。【有伴侶者】在關係中...(1-2句)。【行動】試著...(1句)。",
```
